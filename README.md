# 样本方差等概率论计算器

## 功能

✔ 均值

✔ 样本方差

✔ 中位数

✔ 百分位数

✔ Excel读取

✔ CSV读取

✔ TXT读取

✔ 表达式解析

✔ 自动绘图

---

## 使用方法

1.
下载Release

2.
打开文件读取.exe

3.
选择从文件读取或者交互性输入

4.
文件读取可以选择Excel、CSV或者txt的数据文件，随后等待一会就会生成计算结果，还能生成对应的柱状图区域，去判断数据的规律。
交互性输入会打开类似控制台的界面，就能跟随提示进行输入，它虽然没有图像能够生成，但是对于少量数据可以实时看到每个阶段的结果和错误。
